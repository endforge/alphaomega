# AlphaOmega

AlphaOmega is a long-term AI learning and engineering project focused on understanding artificial intelligence from first principles while building a production-quality Personal Knowledge Platform.

## Objectives

- Learn AI deeply rather than simply use AI tools.
- Build a Personal Knowledge Platform.
- Build an AI Creative Collaborator.
- Organize personal Sources of Truth.
- Develop practical AI-powered tools.

## Current Status

AlphaOmega is currently progressing through a series of engineering labs that teach the foundational components required to build a Retrieval-Augmented Generation (RAG) system and, ultimately, a complete AI knowledge platform.

## Repository Philosophy

This repository contains the production implementation of AlphaOmega.

Generated artifacts, embeddings, corpora, and other derived data are intentionally excluded from version control. OneDrive and OneNote remain the authoritative Sources of Truth.
